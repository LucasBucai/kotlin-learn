# Misión de aprendizaje: Kotlin

Quiero aprender Kotlin para mejorar como desarrollador mobile/backend y entender mejor el ecosistema Android/Kotlin.

## Objetivo principal

Aprender Kotlin desde una base práctica, comparándolo cuando sirva con lenguajes que ya conozco, especialmente Dart/Flutter.

## Enfoque

- Aprender con ejemplos simples.
- Practicar escribiendo código.
- Entender sintaxis, null safety, clases, funciones, colecciones, coroutines y arquitectura.
- Evitar teoría innecesaria.
- Priorizar casos reales de desarrollo.

## Nivel actual

Sé programar y tengo experiencia en Flutter/Dart, pero estoy empezando con Kotlin.
